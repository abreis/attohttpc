# Goals
* Light-weight
* Synchronous
* Secure
* Easy to use

# Non goals
* High performance
* Asynchronous

## Features
* Adding query parameters
* Modifying headers
* Tls support
* Automatic redirection
* Streaming response body
* Text encoding support
